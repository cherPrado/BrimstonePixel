## Getting Started

1. Clone the repository to your local machine

```bash
    git clone git@github.com:cherPrado/brimstone-pixel-projeto-pessoal.git
```

2. Install the dependencies

```bash
    npm install
```

3. Start the application

```bash 
    npm run dev
```

4. Open in browser
```bash 
    http://localhost:3000/
```
